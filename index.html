Fabulous.js

Lots and lots and lots of people select text while reading it on the web. You're probably doing it right now. But a blue background color is just too boring for the modern web experience. Your readers want novelties. They want pretty things. Do they want rainbows? Let's assume they do. This is where Fabulous.js comes in.

What?

Fabulous.js turns selected text into a rainbow.

It is a jquery plugin.

Yes, it is silly.

No, you probably shouldn't use it.

It only really works well on websites with darker backgrounds.

It can make your content feel… what's the word I'm looking for… fantastic? Terrific? Astonishing? It's right on the tip of my toungue…

Take Heed

Text selection behaviour in browsers is a non-standard feature. It has been removed from the CSS3 specification and is unlikely to be a part of CSS4. Despite that, it works in all major browsers except IE 7/8.

By default, Fabulous.js is very naughty and disables the selection styles of all elements not rainbowed by Fabulous.js—The text is still selected, but there is no visual feedback—which could massively annoy some of your users. Pass false to the disableOtherSelectionStyles option to turn off this behaviour.

Getting Started

Fabulous.js is a jQuery plugin, so make sure you include jQuery in your page before the jquery.fabulous.js file.

Example: If the main content of your page is wrapped in <section> tags: 

$("section").fabulous()

If you want all elements highlighted in color: 

$("body").fabulous()

The plugin may be called multiple times on the same page with different selections, as demonstrated below.

Options

$(selection).fabulous(options)

Options are passed to Fabulous.js through an object literal.

# style (String)

"cubehelix-rainbow" (Default): The "Less-Angry Rainbow" color scale by Mike Bostock, inpired by Matteo Niccoli's perceptual rainbow.

"rainbow": A standard cycling of hues. Very pretty, but the yellows are harder to read.

"pride": The LGBT movement flag. This forces the cycle option to 6 so that the correct colors are maintained.

# cycle (Number)

The wavelength of the rainbow, as measured in the number of elements. Use a bigger number (~100) to have a smooth color transition, and a smaller number to cycle through a few colors quickly.

# rotation (Number)

Rotate the rainbow by a given amount.

# glow (Boolean)

Adds a glow to the highlighted text. WebKit browsers allow selection text-shadow to extend beyond the bounds of the text, so the glow amount is quite large. Firefox doesn't, so the glow amount is quite small.

# disableOtherSelectionStyles (Boolean)

Adds a blanket rule to all non-Fabulous'd elements that disables selection highlight. It doesn't actually disable selection, but only makes it look that way.

True (Default)

False

# preview (Boolean)

Generates a nice little preview of the selection style in console.log.

# remove (jQuery selection)

Doesn't apply fabulous.js to the passed selection or its children.

In this documentation, it's used to separate the main rainbow from the documentation examples.

# Lettering.js

Lettering.js lets you split up text into many smaller elements. If you want a sweet-as title like at the top of this page, include jquery.lettering.js in your application, and call lettering() and then fabulous() on your jquery selection. 

$("h1").lettering().fabulous()
