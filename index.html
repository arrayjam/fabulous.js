Fabulous.js

Lots and lots and lots of people select text while reading it on the web. You're probably doing it right now. But a blue background color is just too boring for the modern web experience. Your readers want novelties. They want pretty things. Do they want rainbows? Let's assume they do. This is where Fabulous.js comes in.

What?

  • Fabulous.js turns selected text into a rainbow.
  • It is a jquery plugin.
  • Yes, it is silly.
  • No, you probably shouldn't use it.
  • It only really works well on websites with darker backgrounds.
  • It can make your content feel… what's the word I'm looking for… fantastic? Terrific? Astonishing? It's right on the tip of my toungue…

Take Heed

Text selection behaviour in browsers is a non-standard feature. It has been removed from the CSS3 specification and is unlikely to be a part of CSS4. Despite that, it works in all major browsers except IE 7/8.

By default, Fabulous.js is very naughty and disables the selection styles of all elements not rainbowed by Fabulous.js—The text is still selected, but there is no visual feedback—which could massively annoy some of your users. Pass false to the DisableOtherSelectionStyles option to turn off this behaviour.

Getting Started

Fabulous.js is a jQuery plugin, so make sure you include jQuery in your page before the jquery.fabulous.js file.

If the main content of your page is wrapped in <section> tags: 

$("section").fabulous()
If you want all elements highlighted in color: 
$("body").fabulous()
The plugin may be called multiple times on the same page with different selections, as demonstrated below.
